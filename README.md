# 🎵 ربات دانلودر موزیک چند پلتفرمی

ربات تلگرامی پیشرفته برای دانلود از پلتفرم‌های مختلف با پنل ادمین کامل!

## ✨ ویژگی‌ها

### 🎵 پلتفرم‌های پشتیبانی‌شده
- **SoundCloud** 🧡 - تک‌آهنگ، پلی‌لیست، آلبوم
- **Spotify** 🟢 - تک‌آهنگ، پلی‌لیست، آلبوم  
- **Deezer** 💜 - تک‌آهنگ، پلی‌لیست، آلبوم
- **YouTube** 🔴 - ویدیو، Shorts، audio only
- **Instagram** 📸 - پست، ریلز، ویدیو
- **TikTok** 🎵 - بدون واترمارک
- **Twitter/X** 🐦 - توییت، ویدیو
- **Pinterest** 📌 - pin، عکس، ویدیو

### 🎯 امکانات کاربر
- 🔍 جستجو در SoundCloud، Spotify، Deezer، YouTube
- ⚙️ تغییر پلتفرم پیش‌فرض جستجو
- 🎼 انتخاب کیفیت (128, 256, 320 kbps + FLAC)
- 📥 دانلود تکی یا کلی پلی‌لیست/آلبوم
- 📊 مشاهده آمار شخصی و سهمیه
- 📱 رابط کاربری با دکمه‌های شیشه‌ای
- 🎁 استفاده از ایموجی‌های پرمیوم تلگرام

### 🛡️ پنل ادمین کامل
- 👥 مدیریت کاربران (بن، آنبن، سطح‌بندی)
- 📊 آمار لحظه‌ای و تاریخی
- 📢 پیام همگانی به همه کاربران
- ⚙️ تنظیمات پویا ربات
- 🔄 مدیریت پروکسی‌ها
- 💰 سیستم سطح‌بندی (عادی، فعال، VIP، ادمین)
- 📝 لاگ تمام اقدامات

### 📊 سیستم سطح‌بندی
| سطح | نام | دانلود روزانه | حجم مجاز |
|-----|-----|---------------|----------|
| 0 | عادی | 5 | 100MB |
| 1 | فعال | 15 | 500MB |
| 2 | VIP | 50 | 2GB |
| 3 | ادمین | ∞ | ∞ |

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
```bash
pip install -r requirements.txt
```

### تنظیمات
1. فایل `config/settings.py` را ویرایش کنید:
```python
# ID تلگرام خود را از @userinfobot بگیرید
ADMIN_IDS = [YOUR_TELEGRAM_ID]

# توکن بات را از @BotFather بگیرید
BOT_TOKEN = "your_bot_token"
```

### اجرای ربات
```bash
python main.py
```

## 📜 دستورات ربات

### دستورات عمومی
- `/start` - شروع ربات
- `/help` - راهنما
- `/search <query>` - جستجو
- `/platform` - تغییر پلتفرم
- `/stats` - آمار شخصی

### دستورات ادمین
- `/admin` - پنل ادمین
- `/ban <id> <reason> [days]` - بن کاربر
- `/unban <id>` - آنبن کاربر
- `/setlevel <id> <level> [days]` - تنظیم سطح
- `/broadcast` - پیام همگانی

## 📝 نکات مهم

1. **قبل از اجرا:** حتماً `ADMIN_IDS` را در `config/settings.py` تنظیم کنید
2. **FFmpeg:** برای تبدیل فرمت‌ها نیاز به نصب FFmpeg دارید
3. **حافظه:** فضای کافی برای دانلود فایل‌های موقت لازم است
4. **اینترنت:** سرعت اینترنت روی عملکرد تأثیر دارد
