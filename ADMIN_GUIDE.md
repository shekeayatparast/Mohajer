# 📋 راهنمای پنل ادمین ربات تلگرام

## 🔐 تنظیمات اولیه

### 1. افزودن ID ادمین
در ابتدای فایل `tgmusicbot.py`، خط زیر را پیدا کنید:
```python
ADMIN_IDS = [123456789]  # Replace with your actual Telegram user ID(s)
```

ID تلگرام خود را جایگزین کنید. برای پیدا کردن ID:
- به ربات [@userinfobot](https://t.me/userinfobot) پیام دهید
- یا از ربات [@getmyid_bot](https://t.me/getmyid_bot) استفاده کنید

### 2. سطح‌های دسترسی

| سطح | نام | دانلود روزانه | حجم روزانه | توضیح |
|-----|-----|---------------|------------|-------|
| 0 | Regular | 5 | 100MB | کاربر عادی |
| 1 | Active | 15 | 500MB | کاربر فعال |
| 2 | VIP | 50 | 2GB | کاربر ویژه |
| 3 | Admin | ∞ | ∞ | ادمین |

## 🎮 دستورات پنل ادمین

### دستورات متنی

#### `/admin` - ورود به پنل ادمین
نمایش منوی اصلی پنل ادمین با دکمه‌های شیشه‌ای

#### `/ban <user_id> <reason>` - مسدود کردن کاربر
```bash
/ban 123456789 ارسال اسپم
```

#### `/unban <user_id>` - رفع مسدودیت
```bash
/unban 123456789
```

#### `/setlevel <user_id> <level> [vip_days]` - تنظیم سطح کاربر
```bash
/setlevel 123456789 2 30    # VIP به مدت 30 روز
/setlevel 123456789 0       # بازگشت به کاربر عادی
/setlevel 123456789 3       # ادمین
```

#### `/broadcast` - پیام همگانی
```bash
# روش 1: ریپلای روی یک پیام
پیام مورد نظر را فوروارد یا ریپلای کنید + /broadcast

# روش 2: متن مستقیم
/broadcast سلام به همه کاربران!
```

## 🎁 ایموجی‌های پرمیوم تلگرام

### ❓ آیا می‌توان از ایموجی پرمیوم در ربات استفاده کرد؟

**پاسخ کوتاه:** بله، اما با محدودیت‌هایی!

### 🔹 روش‌های استفاده:

#### 1. **استفاده مستقیم (ساده‌ترین روش)**
کافیست ایموجی پرمیوم را در متن کپی کنید:
```python
bot.send_message(chat_id, "✨🎉🌟💎🔥")
```

**لیست ایموجی‌های پرمیوم:**
- ✨ (Sparkles)
- 🎉 (Party Popper) 
- 🌟 (Glowing Star)
- 💎 (Diamond)
- 🔥 (Fire)
- ❤️‍🔥 (Heart on Fire)
- 🫡 (Saluting Face)
- 🫣 (Peeking Eye)
- 🫠 (Melting Face)
- 🫨 (Shaking Face)

#### 2. **Custom Emoji API (پیشرفته)**
تلگرام API جدیدی برای ایموجی‌های سفارشی دارد:

```python
from telebot.types import MessageEntity

# ارسال با custom emoji
bot.send_message(
    chat_id,
    "سلام با ایموجی خاص!",
    entities=[
        MessageEntity(
            type="custom_emoji",
            offset=0,
            length=2,
            custom_emoji_id="1234567890ABCDEF"
        )
    ]
)
```

**دریافت custom_emoji_id:**
```python
# دریافت لیست ایموجی‌های موجود
stickers = bot.get_custom_emoji_stickers(["✨"])
for sticker in stickers:
    print(sticker.custom_emoji_id)
```

### ⚠️ محدودیت‌ها:
1. **کاربران معمولی** ایموجی‌های پرمیوم را می‌بینند اما نمی‌توانند ارسال کنند
2. **ربات‌ها** می‌توانند هر ایموجی پرمیومی را ارسال کنند
3. برای **custom emoji** نیاز به `custom_emoji_id` دارید
4. برخی کلاینت‌های قدیمی تلگرام ایموجی‌های پرمیوم را نشان نمی‌دهند

### ✅ بهترین روش برای ربات شما:
```python
# در متن‌های خوش‌آمدگویی و پیام‌های ادمین
WELCOME_TEXT = """
✨🌟 به ربات ما خوش آمدید! 🌟✨

💎 ویژگی‌های ویژه:
🔥 دانلود سریع
🎉 پشتیبانی عالی
❤️‍🔥 کیفیت بالا
"""

bot.send_message(chat_id, WELCOME_TEXT, parse_mode="HTML")
```

## 🚀 نکات مهم

1. **قبل از اجرای ربات:**
   - ID ادمین را تنظیم کنید
   - دیتابیس به‌طور خودکار جداول جدید را می‌سازد

2. **برای تست:**
   ```bash
   python3 tgmusicbot.py
   ```

3. **مشاهده لاگ ادمین:**
   ```sql
   SELECT * FROM admin_logs ORDER BY timestamp DESC LIMIT 20;
   ```

4. **مشاهده کاربران VIP:**
   ```sql
   SELECT u.chat_id, l.vip_until 
   FROM user_levels l 
   JOIN users u ON l.chat_id = u.chat_id 
   WHERE l.level = 2;
   ```
